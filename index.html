Unit 1: Object-Oriented Programming

Introduction to OOP

Object-Oriented Programming is a programming paradigm based on objects. It focuses on data rather than functions and improves reusability, security, and maintainability.

Class and Object

A class is a blueprint for creating objects. An object is an instance of a class and occupies memory.

Encapsulation

Encapsulation binds data and functions together and hides internal details using access specifiers.

Abstraction

Abstraction shows only essential features and hides implementation details.

Inheritance

Inheritance allows one class to acquire properties of another, promoting code reuse.

Polymorphism

Polymorphism allows one interface to have multiple forms.

Unit 2: Constructors and Destructors

Constructor

A constructor is a special member function automatically called when an object is created.

Types of Constructors

Default, parameterized, copy, and dynamic constructors are used for initialization.

Destructor

A destructor releases memory and is called when an object is destroyed.

Unit 3: Inheritance and Polymorphism

Inheritance

Inheritance allows derived classes to acquire properties of base classes.

Types of Inheritance

Single, multiple, multilevel, hierarchical, hybrid inheritance.

Polymorphism

Polymorphism allows same function names to behave differently.

Virtual Functions

Virtual functions support runtime polymorphism.
